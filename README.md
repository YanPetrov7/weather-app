# Weather App
GitHub Pages: [*link*](https://yanpetrov7.github.io/weather-app/)
## Installation
1. Clone the repo
```sh
git clone git@github.com:YanPetrov7/weather-app.git
```
2. Move to the directory
```sh
cd weather-app
```
3. Install NPM packages
```sh
npm install
```
4. Run the app
```sh
npm start
```
